# 🎓 simple_student_management

**A no-frills student management system — add, view, and check results in seconds.**

A menu-driven Python program that manages student records entirely in memory. Comes preloaded with 30 sample students, and lets you add new ones, view the full list, or instantly check if a student has passed or failed.

---

## ⚡ Why This Exists

Every CS student eventually builds a "student management system" as a rite of passage — it's the perfect project to practice dictionaries, loops, and conditional logic. This one keeps it clean and practical: no databases, no files, just straightforward Python doing exactly what's needed.

---

## 🚀 Features

- ➕ **Add a student** — name and marks, added instantly to the record
- 📋 **View all students** — full list with names and marks, printed cleanly
- ✅❌ **Check result** — automatically determines PASS/FAIL based on a 40-mark threshold
- 🗂️ **30 preloaded sample students** — ready to explore right out of the box
- 🔁 **Menu-driven loop** — keeps running until you choose to exit

---

## 🛠️ Usage

```bash
git clone https://github.com/deepanshu-bisht-dev/simple_student_management.git
cd simple_student_management
python main.py
```

You'll see a simple menu:

```
------STUDENT MANAGEMENT------
1. ADD STUDENT
2. VIEW STUDENT
3. CHECK RESULT
4. EXIT
```

Just type the number of what you want to do, and follow the prompts.

## 📦 Tech Stack

- **Language:** Python 3
- **Data structure:** Dictionary (in-memory, no external storage)
